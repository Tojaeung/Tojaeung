# Project
[그림러들 - 그림 그리는것을 좋아하는 사람들의 모임](https://grimlerdl.com)

* 개발기간: 2022년 1월 ~ 5월
* 운영기간: 2022년 6월 ~

저의 주변에 그림을 잘 그리는 친구가 어떤 네이버밴드 모임에서 자기가 그린 그림을 공유하고 소통하고 있었습니다. 그 모임 나름대로 그림에 관련한 은어와 문화, 규칙들을 가지고 있는게 흥미롭게 느껴졌습니다. 저는 네이버밴드라는 플랫폼 한계에서 벗어나 오직 이 모임을 위한 커뮤니티 웹 서비스를 프로젝트로 만들어봐야겠다고 결심하였습니다. 그리고, React,Typescript를 사용해서 그림 커뮤니티 게시판 프로젝트를 완성하였습니다. 사실, 네이버밴드 플랫폼 한계를 뛰어넘겠다는 처음의 포부와 다르게 저의 프로젝트는 많이 부족합니다. 프로젝트를 진행하면서 개발 초보자인 제가 각 분야 전문가가 모여서 만든 네이버밴드를 능가하는것은 불가능하다고 느꼈습니다. 저의 프로젝트(그림러들)의 특징은 위에서 언급한것과 같이 실제 사용자가 사용한다고 생각하고 개발을 하였습니다. 그래서, 이메일 인증, 프로필 변경 기능 등과 같이 실제 서비스와 유사하게 만들기 위해 노력하였습니다. 그리고, 채팅,알림, 신고기능 등 다양한 기능을 구현하였습니다. 저의 프로젝트는 아래의 링크에 추가하였습니다.  
[프로젝트 확인하러가기](https://github.com/Tojaeung/web-character-project)
